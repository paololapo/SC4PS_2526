# Exercise 1 — Coin Tosses and Law of Large Numbers

## Goal

Generate 100,000 fair coin tosses using an LCG random number generator and study the running fraction of heads.

For a fair coin, the expected probability of heads is:

\[
P(\text{heads}) = 0.5
\]

By the Law of Large Numbers, the running fraction of heads should approach 0.5 as the number of tosses increases.

---

## Files

- `coin_toss_lln.c`: C program
- `plot.py`: Python plotting script
- `coin_toss_lln.txt`: data file generated by the C program
- `coin_toss_lln.png`: plot generated by Python

---

## Compile

```bash
gcc -std=c11 -Wall -Wextra -O2 coin_toss_lln.c -o coin_toss_lln
```

---

## Run the C program

```bash
./coin_toss_lln
```

This creates:

```text
coin_toss_lln.txt
```

The file contains:

```text
toss_index    running_fraction_of_heads
```

---

## Make the plot

```bash
python3 plot.py
```

This creates:

```text
coin_toss_lln.png
```

---

## Expected behavior

At the beginning, the fraction of heads can fluctuate a lot.

As the number of tosses grows, the running fraction becomes more stable and approaches 0.5.

This is an example of the Law of Large Numbers.
